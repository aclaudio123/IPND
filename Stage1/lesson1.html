Important Concepts

Lesson 1: The Basics of the Web and HTML

How the Web Works

The web is basically a bunch of computers that communicate with each other. For example, when a person goes to a web page like www.google.com and request an information, their computer sends an HTTP Request to a Server. HTTP stands for HyperText Transfer Protocol which is the main protocol of the web. Server is the name of computers that host HTML documents. Therefore, the server finds the appropriate HTML document and sends it back to the user's computer where a web browser like Internet Explorer, Safari, FireFox or Chrome interprets the page and displays it on the user's screen. This video does a good job of explaining.

HTML Basics

HTML stands for HyperText Markup Language. It is made up of text content which is "what you see" and markup which is "how it looks". HTML documents are the heart of the web and form the majority of the content. They also make references to other documents, images, videos, and links to other pages on the web. This video gives a good overview.

Tags and Elements

HTML markup is made up of tags and content. Content is what goes inside the openning and closing tags. Examples of tags are <b>, <em>, <a>, and <img>. When we put any content inside an openning and closing tag, that whole structure is what we call an HTML element. For example <b>content</b> and <em>content</em> are both HTML elements.
We use HTML tags and elements to tell a web browser how to display content on a web page. This video and this video show how HTML tags and elements are used.

Note: Always remember to open and close HTML tags appropriately else the content inside of them and subsequent contents won't display like you wanted them to. That, explains why computers are stupid.

Why Computers are Stupid

The truth is computers are stupid because they interpret instructions exactly the way you enter them. They are very sensitive to typos and as a result, small mistakes by a programmer can cause huge problems in a program. This article explains it better.

HTML Attributes

Some HTML tags accept additional values called attributes. Attributes provide additional information to the browser. For example,

  1. The anchor tag <a> takes an attribute called href that defines a URL to link to. The tag structure below creates a hyper link to the href attribute "url" and names it link.
    <a href="url">link</a>
  2. The <img> tag structure <img src="url" alt="text"> displays the image hosted at the src attribute "url". The url can also be a path to a local image file on your computer. If the image link is broken and can not be displayed, the alt attribute "text" will be displayed on the screen. Remember "text" is what you choose to display on screen.

FYI - Notice the difference between the <a> and <img> closing tag. You should see that the <img> tag doesn't have a closing tag with a backlash and the tag name. That is because the <img> tag has no content inside of it whereas the <a> tag had content which was the word link. Any element without the closing tag is called a void element.

Inline vs Block Elements

All HTML elements can be categorized into two categories:

  1. Inline Elements

    Inline elements end a line of text and wrap new text to the next line. They can appear within sentences and do not have to appear on a new line of their own. For example <b>, <i>, <u>, <em>, <strong>, <sup>, <sub>, <big>, <small>, <li>, <ins>, <del>, <code>, <cite>, <dfn>, <kbd>, and <var> are all inline elements.

  2. Block Elements

    Block elements make an "invisible box" around the content inside of them. They appear on the screen as if they have line breaks before and after them. For example the <p>, <h1>, <h2>, <h3>, <h4>, <h5>, <h6>, <ul>, <ol>, <dl>, <pre>, <hr />, <blockquote>, and <address> are all block elements.

To learn more about inline vs block elements, this video explains the difference.

Note: For a browser to be able to display the less than (<) and greater than (>) symbols before and after an HTML tag, you have to use html-escape characters.

Grouping HTML Elements

There are two very important tags that are used to group various other HTML tags. They are the <span> and <div> tag. The <span> tag is use for grouping and applying style to inline elements whereas the <div> tag is use for grouping and applying style to block elements.
